# Privacy And Asset Hygiene

This project is meant to stay local-first.

- Wallpapers are not included in the repository.
- The patcher stores local theme state next to the installed copy of the tool.
- Generated previews, backups, logs, and extracted app files should not be
  committed.
- Issue reports should redact local paths, account names, tokens, and private
  filenames.
- Do not upload copyrighted wallpapers unless you have permission to share them.

The repository `.gitignore` excludes common local output folders such as
`generated/`, `backups/`, `theme-tests/`, logs, and local `theme.json` files.
